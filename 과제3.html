CSS 2D Transform

[메뉴로]

The translate() Method:이동 메소드

translate()있어서 (상기 X 축 및 Y 축에 주어진 파라미터에 따라) 현재 위치에서 요소를 이동시킨다.
예시
div {transform: translate(50px, 6px);}
실행결과
오른쪽으로 50px만큼 아래쪽으로 6px만큼 이동함.

The rotate() Method:회전 메소드

rotate()방법은 소정의 정도에 따른 소자를 시계 방향 또는 시계 반대 방향으로 회전
예시
div {transform: rotate(10deg); /*시계방향으로 10도 회전*/}
div {transform: rotate(-10deg); /*반시계방향으로 10도 회전*/}

실행결과
시계방향으로 10도 회전하였습니다.
반시계방향으로 10도 회전하였습니다.

The scale() Method:규모 메소드

scale()있어서 증가하거나 (폭 및 높이에 대한 특정 매개 변수에 따라) 요소의 크기를 감소
예시
div {transform: scale(2, 3); /*폭 2배, 높이 3배*/}
div {transform: scale(0.5, 0.5); /*폭,높이 절반*/}
실행결과
폭 2배, 높이 3배로 증가 하였습니다
폭과 높이가 절반으로 감소 하였습니다

The skewX() Method, The skewY() Method : skewX () 메소드, skewY () 메소드

X또는 Y 축을 따라 요소 기울임
예시
div {transform: skewX(20deg); /*X축을 기준으로 20도 기울임*/}
div {transform: skewY(20deg); /*Y축을 기준으로 20도 기울임*/}
실행결과
x축을 기준으로 20도 기울임
y축을 기준으로 20도 기울임

skew() Method:skew(스큐)메소드

skew()로 X및 Y 축을 따라 요소를 기울임
예시
div {transform: skew(20deg, 10deg); /*X축으로 기준으로 20도, Y축을 기준으로 10도 기울임 두 번쨰 매개변수를 입력하지 않으면 X축으로만 기운다.*/}
실행결과
The skew() method skews an element into a given angle.

The matrix() Method:매트릭스 메소드

matrix()를 사용하는 방법은 모든 2D 하나로 변환 방법을 결합, 메소드는 수학가 회전 할 수 있도록 기능, 크기, 이동 (변환)를 함유하는 여섯 개 매개 변수를 사용하고, 요소 스큐.
파라미터 : 앞쪽부터 scaleX () skewY () skewX () scaleY를 () translateX () translate Y() 이다.
예시
div {transform: matrix(1, -0.3, 0, 1, 0, 0);}
실행결과

How to Use CSS Transitions? : 어떻게 CSS 전환을 사용할 수 있나요?

어떻게 CSS 전환을 사용할 수 있나요?
당신이 원하는 CSS의 속성 효과를 추가 효과의 지속 시간 시간을 지정하지 않으면 기본 값이 0이기 때문에
전환이 아무런 영향을 주지 않습니다.
예)) 100 px * 100 px 빨강
요소를 나타낸다.
약 3 초 동안에, 가로로 늘어나는 전환 효과를 지정함:
예시

Specify the Speed Curve of the Transition : 전환 속도 조정

transition-timing-function속성 : 특정 애니메이션을 실행할 때 애니메이션의 가속운동, 즉 특정한 시점에서 가속할지 감속할지를 정하는 속성
속성
  • ease - 느린 시작과 전환 효과를 지정하고 빠르게, 다음 (이것이 기본값입니다) 천천히 종료
  • linear - 처음부터 끝까지 같은 속도로 전환 효과를 지정
  • ease-in - 느린 시작과 전환 효과를 지정합니다
  • ease-out - 느린 엔드와 전환 효과를 지정합니다
  • ease-in-out - 느린 시작과 끝이있는 전환 효과를 지정합니다
  • cubic-bezier(n,n,n,n) - 당신은 입방 베 지어 함수에서 자신의 값을 정의 할 수 있습니다.
    (P0는 (1, 1) 최종 시간 및 최종 상태를 나타내고, P3이, (0, 0)의 초기 시간 및 초기 상태를 나타낸다.)
#div1 {transition-timing-function: linear;}
#div2 {transition-timing-function: ease;}
#div3 {transition-timing-function: ease-in;}
#div4 {transition-timing-function: ease-out;}
#div5 {transition-timing-function: ease-in-out;}
예시
div {width: 100px;height: 100px;background: red;transition: width 2s;transition-timing-function: cubic-bezier(0.1, 0.7, 1.0, 0.1);}
